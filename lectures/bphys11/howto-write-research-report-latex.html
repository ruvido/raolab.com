HOWTO write a research report in latex

The easiest way to write a clean and neat research report is to use the [latex] typesetting system. This is a document markup language , meaning that you will write in your document some commands that will be lately interpreted by a specific program. At the end what you get is a pdf file. For example, suppose you wrote the report in a file called "myreport.tex" you will get the corresponding pdf (that is what you will give me), by running in the terminal the following command:

pdflatex myreport.tex > myreport.pdf

In scientific writing, the use of latex gives several advantages, including: automatic formatting, automatic section and figure numbering and clean equation insertion.

To start to write your first document in latex, open an empty file into your text editor of choice (e.g. gedit). First you need a preamble. This is a set of commands that will be interpreted by the latex program at the compiling stage (i.e. pdflatex ...). In the preamble, some variables are defined as well as the use of some special packages. To write a report use the following:

\documentclass[a4paper,11pt]{article} \usepackage{graphicx}

The text of the report starts after the following command:

\begin{document}

First put the title, name, group and date of the project:

\title{\sf Allosteric control in Hemoglobin}
\author{GR0: F. Rao, D. Prada-Gracia}
\date{\today}
\maketitle

Each new section starts with \section command, for example:

\section{introduction}

a figure is introduced with:

\begin{figure}
\centering
\includegraphics[width=80mm]{myfig}
\caption{Put here a description of the figure}
\label{mydata}
\end{figure}

the label command is useful when recalling a figure within the text using, for example \ref{mydata} . When using pdflatex (like here), the file "myfig" can be either a pdf, jpg or png image. The document is finished by:

\end{document}
