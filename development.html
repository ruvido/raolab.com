WORDOM is the trajectory analysis tool for the command-line eager. Conceived to spare the user some time in manipulating and converting dcd, xtc, pdb, crd and xyz files, it is also a versatile program for a broad range of molecular structure and dynamics analysis.

Go to Sourceforge to download the latest version »

AQUA-LAB is an in-house developed program for studying water structure and dynamics in biological environments. It is a python-based library with fortran routines to speed-up the most intensive calculations. Highlights include:

  • Open source
  • Complex network analysis
  • Markov-state-models based on local fluctuations analysis
  • Water microstate definitions
  • Global and microstate-based radial distribution functions
